These are some additional files to run the program.  
1. open your terminal in this folder and run :
   ros2 launch webots_ros2_mavic robot_launch.py
2. open a second terminal and run : 
   python3 Gpt.py
3. on this terminal there is an IP address to open the UI to enter prompts via the webserver 
4. open the third terminal and run :
   python3 Gpt_parser.py
